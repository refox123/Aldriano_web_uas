Selamat Datang di Dunia Kreatif Saya

Saya adalah seorang pengembang game dan animator yang bersemangat menciptakan pengalaman digital yang menarik. Portfolio ini menampilkan berbagai projek dan hobi saya dalam dunia game dan animasi.

Tentang Saya

Halo! Saya seorang pengembang game dan animator dengan pengalaman lebih dari 5 tahun di industri kreatif. Saya memiliki passion yang besar untuk menciptakan pengalaman interaktif yang memukau melalui kombinasi teknologi dan seni.

Spesialisasi saya meliputi pengembangan game menggunakan Unity dan Unreal Engine, serta pembuatan aset 3D dan animasi karakter. Saya selalu bersemangat untuk mempelajari teknologi baru dan menerapkannya dalam pekerjaan saya.

Selain bekerja profesional, saya juga aktif dalam komunitas pengembang game lokal dan sering berbagi pengetahuan melalui workshop dan tutorial online.
